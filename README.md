Back-end for [wesequence](https://github.com/jurelik/wesequence).

To use:
```
npm install
```
Create .env.development and .env.production files with the following variables:
```
PORT=
DB_PORT=
DB_USER=
DB_URL=
DB_NAME=
DB_PASSWORD=
```
```
npm run dev
```

This repo requires a running Postgres database instance.
